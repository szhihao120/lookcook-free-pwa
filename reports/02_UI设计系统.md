# LookCook UI 设计系统

## 设计方向

关键词：温暖、可信、食欲感、轻松、非科技炫技。界面避免“AI 仪表盘”风格，让用户
进入页面后第一注意力始终是“家里有什么、今晚吃什么”。

## 视觉令牌

| 令牌 | 值 | 用途 |
|---|---|---|
| `--paper` | `#fffaf2` | 米白页面底 |
| `--surface` | `#ffffff` | 卡片与输入框 |
| `--ink` | `#29251f` | 主文字 |
| `--muted` | `#7b746a` | 次要信息 |
| `--accent` | `#f45d3f` | 品牌珊瑚红 |
| `--green` | `#52765a` | 隐私、在线与健康提示 |
| `--radius` | `26px` | 主要卡片圆角 |
| `--shadow` | 柔和棕色 9% 阴影 | 卡片层级 |

字体仅使用系统中文字体：Microsoft YaHei、PingFang SC、系统 sans-serif；不请求
Google Fonts 或其他外部字体。

## 组件

- 品牌标记：渐变珊瑚红圆角方块和 `L` 字标；
- 主按钮：高对比珊瑚红渐变，最小移动端高度 57px；
- 菜谱卡：原创 SVG 封面、时间、标签、收藏；
- 食材卡：emoji 仅作辅助，文字是无障碍名称；
- 搜索框：菜名、别名、食材、全拼、首字母统一入口；
- 底部导航：移动端四入口；桌面改为顶部导航；
- Service Worker 提示：位于固定操作按钮之上，避免遮挡。

## 响应式

已验证 360×800、390×844、430×932 和 1440×1000，无水平溢出。

- 小于 820px：移动布局、底部导航、两列菜谱；
- 小于 390px：收藏列表单列、设置卡纵向；
- 桌面：1180px 内容宽度、三列菜谱、完整摘要。

## 无障碍

- 语义化 `header/main/nav/article/section`；
- 图片提供明确替代文本；
- 所有图标按钮提供 `aria-label`；
- 支持 `prefers-reduced-motion`；
- 修复搜索数量标签和更新按钮对比度后，Lighthouse Accessibility 为 100。
