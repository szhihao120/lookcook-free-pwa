## 已修复缺陷

1. pnpm 11 默认阻止 esbuild 安装脚本：以 allowBuilds 明确批准。
2. vite-plugin-pwa 缺少显式 workbox-window：补充运行依赖。
3. 移动底栏遮挡推荐按钮：提高固定按钮底部安全区。
4. 离线就绪提示遮挡固定按钮：提示上移。
5. 多次构建残留旧 hash 资源：新增限定路径的构建清理脚本。
6. 两处颜色对比不足和缺 robots.txt：修复后无障碍/SEO 均为 100。
7. GitHub Pages 不提供 SPA rewrite：切换为 Hash 路由，直接刷新详情链接不再 404。

## 未完成的现场证据

Windows 自动化无法确认真实 Chrome 当前 URL，按安全要求停止了最终“安装”点击。
这不影响 Web App 的可安装技术条件，但正式发布验收建议由用户在部署后的 HTTPS
Pages 地址上安装一次并补一张系统级截图。
